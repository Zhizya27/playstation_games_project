# PlayStation Sales Analytics (PS3/PS4/PS5)

Pet-проект по исследованию данных о продажах игр для консолей PlayStation (PS3/PS4/PS5) и построению интерактивного дашборда на базе  Preset/Superset.

- **Источник данных:** Kaggle — Playstation Sales and Metadata (PS3 / PS4 / PS5)  
- **Цель:** превратить сырой датасет в удобный аналитический слой и визуальный дашборд для изучения продаж игр по платформам, периодам и другим параметрам.
- **Стек:** Python (Google Colab, pandas, matplotlib, seaborn, scipy, numpy), Superset.

> Google Colab ноутбук: https://colab.research.google.com/drive/1W5J6cOjZwB38FKVccqjt6OjESDRV1kiT?usp=sharing   
> Датасет Kaggle: [Playstation Sales and Metadata (PS3 / PS4 / PS5)](https://www.kaggle.com/datasets/gvidalguiresse/playstation-sales-and-metadata-ps3ps4ps5)


## Цели проекта

- Отработать полный цикл работы с данными:
  - Загрузка сырого датасета из Kaggle.
  - Очистка и предобработка данных.
  - Исследовательский анализ (EDA) и визуализация в Python.
  - Подготовка итогового датафрейма и экспорт в CSV.
  - Построение интерактивного дашборда в Superset.


## Стек и инструменты

- **Python / Google Colab**
  - Работа с данными и EDA
  - Очистка и преобразование датасета
  - Создание визуализаций для первичного анализа
- **Preset/Superset**
  - Подключение к итоговому CSV как к источнику данных
  - Настройка метрик и фильтров
  - Построение интерактивного дашборда

## Обработка данных и EDA (Python / Colab)

Основные шаги, которые выполняются в ноутбуке:

1. **Загрузка данных**
   - Импорт исходного CSV-файла из Kaggle
2. **Очистка данных**
   - Работа с пропусками (удаление/заполнение)
   - Приведение типов (даты, числовые поля)
   - Нормализация и переименование столбцов, где это необходимо
3. **Добавление новых признаков**
   - Создание дополнительных столбцов, упрощающих аналитику  
     (например,категории или группировки)
4. **EDA и визуализации**
   - Базовые описательные статистики
   - Гистограммы, барчарты и другие графики для анализа:
     - распределения продаж
     - сравнения платформ
     - сравнения ключевых категорий (н-р, жанры, издатели)
5. **Экспорт итогового датафрейма**
   - Сохранение обработанного датафрейма в CSV для дальнейшего импорта в Superset


## Дашборд в Superset

После завершения обработки данных:

1. Итоговый CSV импортируется в Superset как источник данных
2. Настраиваются измерения и метрики
3. На основе источника создаётся интерактивный дашборд

Посмотреть получившийся дашборд можно в папке dashboard, файл **dashboard_playstation_games_project.jpg**


Чтобы импортировать дашборд в Superset, скачайте **zip-файл dashboard_15_example.zip**
